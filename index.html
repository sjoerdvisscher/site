Typedefs

Language agnostic definition of types and type constructors.

Minimal "type system", modelled after polynomial functors and W-types

Used by statebox

See github/typedefs

example

type foo : *;
type Ints = array i32;

type Units : *;
type Units = array product { };

type List : * -> *;
type List = array;

type Zero : *;
type Zero = sum { };

type One : *;
type One = product { };

type Maybe : *;
type Maybe = sum { just: i32, nothing: One };

/* comment */

type EmptyArray = array sum { };
